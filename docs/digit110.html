DIGIT 110: Text Encoding Fundamentals

I took DIGIT 110 in Fall 2020 when I was a Sophomore. In this class, I learned many things including XML, Regex, RelaxNG, HTML, CSS, and more. Our main project for the class was a group project where we formed teams to markup a document and make a digital version of it. My project group marked up a collection of poems written by Langston Hughes titled "Montage of a Dream Deferred."


We worked on most of the project together, but the main parts I contributed to were the original XML markup and schema creation, some CSS work, and the co-creation of the "About the Project" page for our website.
